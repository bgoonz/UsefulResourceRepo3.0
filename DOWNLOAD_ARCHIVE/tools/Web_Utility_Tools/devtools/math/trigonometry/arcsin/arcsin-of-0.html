What is the arcsine of 0 ?

arcsin 0 = ?

The arcsine is the inverse sine function.

Since

sin 0 = sin 0º = 0

The arcsine of 0 is equal to the inverse sine function of 0, which is equal to 0 radians or 0 degrees:

arcsin 0 = sin-1 0 = 0 rad = 0º
