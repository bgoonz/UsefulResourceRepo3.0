    UNIT 1

    Data Analytics Foundations

    Launch your data analytics journey with an introduction to data by learning foundational concepts and new skills for approaching data problems. You'll start your learning journey by developing a problem-solving framework and by understanding how data should be collected and prepared for data analysis. Translate your learnings into job-ready skills by solving real-world problems with a variety of data sets.

    • Microsoft Excel
    KEY SKILLS:
    Analysis Strategy
    Data Problem Solving
    Data Collection
    Data Cleaning
    Advanced Excel

    Learn Problem-Solving Strategies for High-Value Outputs

    Set yourself up for success by approaching data analysis correctly. Learn data analysis and apply strategic principles with an interactive problem-solving methodology to analyze data.

    Source High-Quality Data

    Learn essential techniques to help you prepare and clean raw data, as well as reduce bias, in order to support effective data analysis. Explore critical statistical analysis concepts used in data analysis.

    Master Excel Data Analysis

    Take your Microsoft Excel skills to the next level, exploring everything from basic formulas to advanced functions and powerful tools that drive efficient analysis and accurate results.

    UNIT 2

    Database Operations & Advanced Data Analytics

    Go beyond the spreadsheet with relational databases. By learning how to use MySQL – an industry-standard tool used by companies like Tesla, Netflix, and more – you'll discover how databases are constructed and organized for data analysis. Learn how to write SQL queries through real-world projects to explore a database and perform more advanced operations and analysis.

    KEY SKILLS:
    Exploratory Data Analysis
    SQL
    Regular Expressions
    Data Management
    Data Normalization
    Database Schema

    Learn Database Fundamentals

    Working with larger data sets and performing more complex data analysis requires an understanding of relational databases, schema, and other core concepts used by data analysts as part of many industry tools.

    Manage and Analyze Data with SQL

    Explore the foundations of programming languages and learn Structured Query Language (SQL) – a powerful programming language and one of the essential programming skills for a career in data.

    Perform Advanced Data Analysis

    Learn how to write advanced expressions to help you surface key data points, highlight patterns, and augment your search capabilities within large data sets.

    Organize and Optimize Data Sets

    Speed up your analysis and reduce complexity by learning a systematic approach to normalize data, along with important techniques for combining and sorting information.

    UNIT 3

    Visualizing & Communicating Data Insights

    Completing your data analysis is only the start. Continue your learning path by discovering how to present actionable insights effectively through data visualizations that inform business decisions created with Tableau, the world's leading business analytics platform. Go beyond simply presenting analysis results by learning best practices in storytelling with data and help drive effective decision-making informed by real data insights.

    • Tableau
    KEY SKILLS:
    Data Visualization
    Dashboards
    Presentation Skills
    Data Communication

    Highlight Critical Data Insights

    Use Tableau – an industry-leading business intelligence and visualization tool – to augment your analysis and distill the most important insights into clear visualizations. Set custom parameters, filtering, and more to make sure the right data from your business analytics stand out.

    Build Impactful Visualizations & Dashboards

    Learn all the skills to build and fine-tune a clear, communicative data visualization, tailored to unique business needs. Apply best practices in hands-on projects to create dynamic and easy-to-use dashboards.

    Communicate Captivating Data Narratives

    Data tells a story – what happened, what can happen, and what to focus on next. Learn how to capture the attention of a broad audience, and create visualizations that effectively communicate the value of your data analysis.

    Frequently Asked Questions

    • What is the difference between data analytics and data science?

      If you want to become a Data Analyst, you should know this is not quite the same thing as becoming a Data Scientist, which is generally a more senior position involving greater technical knowledge and data science expertise. In fact, Data Analyst positions, and data analysis work, are often considered more entry-level in nature.

      BrainStation's Data Analytics course is a beginner-friendly data course, with a focus on descriptive analytics, industry standard tools and software. The Data Science course, on the other hand, can be considered more of an intermediate-level data science course, as it is more technically demanding, diving further into prescriptive and predictive analytics, machine learning, and programming languages.

    • Will I earn a data analytics certificate from this course?

      Yes, when you complete the Data Analytics course, you will earn a BrainStation Data Analytics Certificate. A data analytics certification can boost your LinkedIn profile and resume, helping you stand out in the job market when applying for data analyst positions and can help you become a data analyst setting you up with the foundational skills and knowledge for a successful career in data.

    • What payment options do you offer?

      BrainStation offers some of the most competitive payment options for data analysis training, with a range of flexible plans and scholarship opportunities.

      These include:

      • Monthly payment installments

        Allowing you to split your tuition into smaller monthly payments.

      • Employer sponsorship

        Get your tuition reimbursed by your employer.

      • Scholarships

        We offer a range of scholarships to make learning data analysis more accessible.

    • Are there any prerequisites for this data analytics course?

      BrainStation's Data Analytics course was created for those who are new to data analysis, and can be considered a beginner-friendly course. There are no set prerequisites for this course, but a working knowledge and prior experience of Excel is strongly encouraged. No other data analytics courses are required before taking this Data Analytics course.

    • What kind of backgrounds do people have that take this course?

      By enrolling in the Data Analytics course, you can expect to learn from an Instructor with experience in the field, and to collaborate with like-minded, ambitious professionals keen to boost their data analysis skills.

      80 percent of learners at BrainStation have a post-secondary education, with 20 percent having earned a Master's degree or PhD.

      Professional backgrounds also vary quite a bit, particularly for the Data Analytics course, as data analysis skills are now highly valued across a range of fields and industries, including finance, marketing, healthcare, retail, and more.

    • What tools and programming languages do we learn?

      Throughout the Data Analytics course, you will learn to work with data more efficiently, starting with taking your Excel skills to the next level through real-world projects. Many of these skills will also apply to Google Sheets.

      You will also learn how to organize data within a database, and access it using Structured Querying Language (SQL). Finally, you will use Tableau to develop high-quality business intelligence dashboards for ongoing monitoring, and present your insights in a meaningful way.

    • What is the difference between this and the Data Science bootcamp?

      BrainStation's Data Analytics course is a flexible, professional development course offered part-time, over 5 or 10 weeks. Taught by industry experts, the Data Analytics course is a project-based, hands-on learning experience, allowing you to develop data analysis skills and learn the latest data tools and technologies.

      BrainStation's Data Science bootcamp, on the other hand, is an intensive learning experience designed to transform your skillset and help you launch a new career in data. Throughout the program, students develop and complete five projects including one major portfolio piece, using real-world data, big data, and machine learning. By the end of the program, graduates will have the skills, experience, and portfolio needed to dive into a career in data.
