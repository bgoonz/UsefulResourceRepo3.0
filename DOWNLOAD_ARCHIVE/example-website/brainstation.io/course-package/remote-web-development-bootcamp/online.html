Program Overview

BrainStation's Online Web Development Bootcamp is an intensive, project-based learning experience, designed to help students launch a new career in web development.

To emphasize collaboration and outcomes-based skills development, the online program provides a low student-teacher ratio, with a comprehensive curriculum that is updated in real-time.

Led by Educators with experience in the field, students will use HTML, CSS, JavaScript, React, Node, and more to gain hands-on experience in full-stack development, including building and deploying multi-page web applications, collaborating with other developers, and working with databases. This knowledge is then applied to complete a range of real-world client projects, in multiple sprints, as well as a final portfolio piece that puts new skills to the test.

By the end of the online program, graduates will have the skills, experience, and portfolio needed to dive into a career in web development.

What You'll Learn

  1. Unit 1
    Web Development Fundamentals – HTML, CSS, Flexbox

    Learn to structure code in a concise, logical, and readable manner, while mastering HTML and CSS, two foundational front-end languages. Create static web pages using HTML and CSS alone, before applying Flexbox (a CSS framework) to increase the complexity and speed in which you create your web pages.

    How you'll apply what you learn:

    Use your new skills to create a professional website that is responsive to both mobile and computer browsers. Your work will be expanded later in the program.

  2. Unit 2
    JavaScript – Functions, Animations, Components

    Sites today don’t just display content – they are interactive, with animations, logins, checkouts, and much more. To add functionality to your websites, you will learn the essentials of object-oriented programming and JavaScript, including how to leverage reusable "building blocks" of code.

    You will then build on these programming fundamentals by learning how to create and implement functions and animations.

    How you'll apply what you learn:

    You will augment your previously developed static site by adding multiple pages with underlying logic to create basic interactive features.

  3. Unit 3
    React – Components, States, APIs

    Leverage React, one of the most popular JavaScript frameworks, to refine the quality of your work and create professional user interfaces. You’ll learn to manage states, allowing your website to work with data in a more dynamic way; and integrate an application programming interface (API), which connects your website to a back-end.

    How you'll apply what you learn:

    Using React, you’ll create a professional single page application, which leverages complex functionality and visual styling.

  4. Unit 4
    Node – Express, Clients

    Learn how websites interact with a back-end to exchange application data. Using Node.js, a JavaScript environment, and Express, a Node framework, you’ll learn how to quickly leverage back-end technologies to create a full-stack application. By understanding how to create your own application programming interface (API), you’ll be able to customize and improve the functionality of your work.

    How you'll apply what you learn:

    You’ll create your own API to add further functionality to front-end applications.

  5. Unit 5
    Agile Development – Git, Pair Programming, Jira

    Understanding how to collaborate with other developers and work on agile teams is a critical skill for all developers. Learn to jointly work on an agile project with other developers, using git to manage versions and share work. You’ll be assigned tasks and use Jira to manage the development process.

    How you'll apply what you learn:

    In a team with fellow online students, you’ll collaboratively build out a web application from start to finish.

  6. Unit 6
    Databases – MySQL, Database Structures

    Understanding different database structures is critical in managing large amounts of information in an efficient and effective way. To store and manage data, you’ll learn how to use MySQL, a high-performance, open-source relational database management system used by data-rich companies like YouTube, Netflix, and Spotify.

    How you'll apply what you learn:

    You will build out the functionality of an application by collecting and storing complex data in your own database.

  7. Unit 7
    Professional Development – Deployment, Authentication, Portfolios

    With a solidified understanding of how to build out web applications, you’ll learn how to deploy your work so that it can be viewed online. You’ll also be introduced to topics such as security, authentication methods, and additional web frameworks to continue to expand your understanding of development.

    How you'll apply what you learn:

    You’ll create a personal, capstone web application that summarizes your learning throughout the program and applies a combination of all tools. We recommend that you deploy this project, along with your personal portfolio site.

Application Process

BrainStation's bootcamps are designed to be challenging, intensive learning experiences. To ensure the success of all our students, BrainStation only selects committed individuals who will thrive in a fast-paced, collaborative learning environment. To be considered for a bootcamp at BrainStation, you must complete the following steps. Please note that the application process can take one to two weeks, on average.

  1. Speak with a Learning Advisor

  2. Submit an Initial Online Application

  3. Receive an Invitation to an Interview

  4. Complete the Admissions Challenge

  5. Acceptance

The BrainStation application process includes:

  • Submission of education credentials

  • Completion of a personal statement

  • Satisfactory completion of BrainStation's digital admissions challenge, which gauges baseline knowledge of the subject matter

  • Satisfactory completion of a live video interview with our admissions team

  • Submission of proof of English language proficiency (if English is not your first language)
