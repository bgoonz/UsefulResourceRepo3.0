cp: omitting directory

cp omitting directory Linux error messge.

Probelm

When trying to copy a directory, you get error message:

$ cp srcdir destdir
cp: omitting directory 'srcdir'
$

Solution

Add -R option for recursive copy:

$ cp -R srcdir destdir
